# IDE 
    P5JS é uma IDE gratuita feita em javascript que facilita a interação com o canvas do HTML e pode rodar em qualquer servidor apache.

# Como usar
    Basta colocar o projeto dentro de um server ou localhost, testado apenas no Apache.
    Acessar o index.html dentro da pasta /platform e ele vai rodar normalmente.
